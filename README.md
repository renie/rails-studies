# RAILS Studies

## System dependencies
	- Ruby 2.4.1
	- Postgres 9.5.8

## Configuration
```
git clone [URL]
cd [clone-path]
gem install bundler
bundle install
```

Create an `.env` file with db usename and password.


## Database creation
```
rails db:migrate
```

## How to run the test suite
```
rails t
```
